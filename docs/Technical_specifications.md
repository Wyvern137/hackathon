ТЗ №1. Генератор контента для НКО
ОБЩИЕ ВВОДНЫЕ

Аннотация
НКО часто не хватает времени или навыков для создания качественного контента. Этот бот поможет генерировать тексты постов, предложения визуалов и идеи контент-плана, обеспечивая лаконичность, читаемость
и соответствие стилю НКО.

Цель проекта
Создать рабочий прототип Telegram-бота, который ускоряет подготовку контента для НКО, делая его понятным для пользователя
и способствующих развитию социальных сетей и сайтов НКО.

Бизнес-задание для участников хакатона
Разработка Telegram-бота для генерации изображения, текста, контент-плана. Бот должен помочь сотрудникам и волонтёрам НКО быстро создавать тексты постов, новости и идеи визуалов для соцсетей. Генерация изображений и текста формируется исходя из данных об НКО, которые бот получает от пользователя после его запуска. Информацию об организации бот обязан получить на основе ответа на вопросы пользователем
(при желании пользователя). На основе этих данных происходит генерация изображения, текста, контент-плана. Исходя из желаемого результата пользователя, бот активирует функции, которые будут полезны
для запросов конкретного НКО.


 ФУНКЦИОНАЛ ДЛЯ ПОЛЬЗОВАТЕЛЯ
Запуск бота и приветствие:
* Пользователь должен понимать, что работа с Telegram-ботом началась и какие возможны дальнейшие действия.
Запрос информации об НКО:
* Бот пошагово задает вопросы об НКО, запрашивая название, описание, формы деятельности и т.д. Пользователь отвечает в текстовом формате. Затем данная информация должна использоваться при генерации контента.
* У пользователя должна быть возможность не предоставлять сведения
об НКО и все равно продолжить работу с ботом. В таком случае
при генерации контента посты должны получаться обезличенными.
Функции бота, которые должны быть доступны пользователю:
* Генерация текста
* Генерация картинки
* Редактор текста
* Создание контент-плана
Функция «генерация текста»
○         Функция «Генерация текста» должна предоставлять возможность пользователю генерировать «свободный текст» (пользователь вводит свою идею, бот «переписывает» её в готовый пост), «структурированная форма» - бот задаёт пошаговые вопросы: что за событие, дата, место, кто приглашён, дополнительные детали,
а также генерация текста, основываясь на примерах
уже готовых постов.
○         Результат: готовый текст для поста без орфографических, грамматических, логических и речевых ошибок, сгенерированный на основе заданных характеристик пользователя. Также пост должен содержать хештег, который отражает тематику поста.
Функция «Генерация картинки».
●         Пользователь вводит описание или прикрепляет свои файлы, которые можно использовать для создания изображений,
бот генерирует картинку через AI и предлагает
её скачать/использовать для поста.
●         Результат: готовая картинка, где визуал соответствует характеристикам пользователя, изображение должно быть качественное, без искажений или нереалистичных деталей.
Функция «Редактор текста»
●         Данная функция помогает пользователю отредактировать
его текст с точки зрения грамматики, орфографии, логики,
речи с предложением советов для улучшения поста.
●         Результат: отредактированный текст пользователя с указанием, где были допущены ошибки и как их исправлять.
Функция «Создание контент-плана»
·      Бот должен предложить контент-план на заданный пользователем промежуток времени, частоту, регулярность.
·      Результат: текстовое сообщение с контент-планом,
где указан день и категория поста для Telegram-канала НКО.
 
ДОПОЛНИТЕЛЬНЫЕ ОПЦИИ
(НЕОБЯЗАТЕЛЬНЫЕ ДЛЯ БАЗОВОЙ ВЕРСИИ)
●         Возможность интегрировать бот в чат-команду НКО
●         Возможность присылать уведомления-напоминания
о публикациях в соответствии с контент-планом

ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ
●         Платформа: Telegram
●         Язык: русский
●         Используемые API: только российские нейросети (например, YandexGPT, GigaChat)
●         Размер текста/картинки: по ограничениям Telegram
 
 РЕКОМЕНДАЦИИ ДЛЯ УЧАСТНИКОВ
●         Минимум кликов — максимум пользы
●         Простой и понятный интерфейс
●         Возможность выбора стиля повествования поста (разговорный/официально-деловой/художественный стиль и так далее)
●         Социально значимая цель: каждая функция помогает НКО быть эффективнее
